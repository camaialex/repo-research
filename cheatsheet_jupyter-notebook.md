# Jupyter Notebook Cheat Sheet

## Startup (Mac)
1. Terminal
2. `% jupyter notebook`
3. `Control^ + C`, `% y` to terminate server connection

## Useful Shortcuts
`Enter` Begin editing selected code block
`Esc` Stop editing selected code block
`D+D` Delete selected code block
`M` Format markdown
`B` Format code
