# Innate Immune System

## Function
- First cellular response to an invading pathogen
	- [[inflammation]]
- Responds generally and in the same way to any perceived threat
- Lacks **immune memory**
## Components
- [[barrier organs]]
- Cellular (cells may fit into multiple groups)
	- [[phagocyte]]
	- [[granulocyte]]
	- [[APC]]
- Noncellular
	- [[complement]]
